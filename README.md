# 🌐 Networking Labs

Repositório dedicado aos meus laboratórios práticos de Redes de Computadores desenvolvidos utilizando o Cisco Packet Tracer.

O objetivo deste projeto é documentar minha evolução na área de redes, desde os conceitos fundamentais até configurações mais avançadas utilizadas em ambientes corporativos e de cibersegurança.

---

## 🎯 Objetivos

- Desenvolver uma base sólida em Redes de Computadores.
- Praticar configurações utilizando Cisco IOS.
- Aprender conceitos de roteamento e switching.
- Construir um portfólio de laboratórios práticos.
- Preparar meus estudos para as áreas de Redes, Cloud e Cybersecurity.

---

## 🛠️ Tecnologias

- Cisco Packet Tracer
- Cisco IOS
- IPv4
- Roteamento
- Switching
